# 언어 및 문서화 규칙

## 응답 언어
- **기본 언어**: 모든 응답과 설명은 한국어로 작성
- **코드 주석**: 가능한 한 한국어로 작성하되, 기술적 정확성 우선
- **기술 용어**: 영어와 한국어 병행 표기 (예: "컨테이너(container)", "비동기(async)")
- **에러 메시지**: 원본 언어 유지, 설명은 한국어 제공

## 코드 작성 규칙
- **변수/함수명**: 영어로 작성 (국제 표준 준수)
- **문서 문자열**: 한국어로 작성하되 docstring은 영어 병행
- **설정 파일**: 키는 영어, 주석은 한국어
- **로그 메시지**: 개발자용은 영어, 사용자용은 한국어

## 예외 상황
- 공식 문서나 명령어는 원본 언어 유지
- 사용자가 명시적으로 다른 언어 요청 시 해당 언어 사용
- 국제적 협업이 필요한 코드는 영어 우선
