# Pawnlib 제품 개요 및 설계 철학

## 핵심 목적
Pawnlib은 Infrastructure as Code(IaC) 운영을 위한 다목적 Python 라이브러리입니다. 개발자 생산성 향상과 개발 워크플로우 간소화를 위한 포괄적인 유틸리티 함수와 클래스를 제공합니다.

## 주요 기능 영역
- **인프라 자동화**: IaC 도구 및 자동화에 중점
- **블록체인 통합**: ICON 네트워크 상호작용 및 모니터링 전문 도구
- **시스템 모니터링**: SSH 로그 모니터링, WebSocket 연결, 리소스 관리
- **개발자 도구**: 반복적인 코딩 패턴 감소 및 코드 가독성 향상

## 핵심 컴포넌트
- **CLI 도구**: `pawns` 명령어를 통한 포괄적인 명령줄 인터페이스
- **HTTP 유틸리티**: 모니터링 및 테스트 기능을 갖춘 고급 HTTP 클라이언트
- **블록체인 도구**: ICON 네트워크 RPC 헬퍼, 지갑 관리, 체인 모니터링
- **시스템 리소스**: 서버 모니터링, 네트워크 유틸리티, 성능 추적
- **비동기 지원**: 고성능 작업을 위한 내장 asyncio 지원
- **Rich 출력**: 예쁜 출력, 콘솔 포매팅, 리치 텍스트 표시

## 대상 사용자
- 인프라 자동화 작업을 하는 DevOps 엔지니어
- ICON 네트워크에서 개발하는 블록체인 개발자
- 일반적인 작업을 위한 유틸리티 라이브러리가 필요한 Python 개발자
- 모니터링 및 자동화 도구가 필요한 시스템 관리자

## 설계 원칙
- **모듈성**: 각 기능은 독립적으로 사용 가능
- **확장성**: 새로운 기능 추가가 용이한 구조
- **사용성**: 직관적이고 일관된 API 제공
- **성능**: 비동기 처리 및 효율적인 리소스 사용
